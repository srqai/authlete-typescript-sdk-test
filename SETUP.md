# Quick Setup Guide for Testing

## For Your Boss / Testers

### Option 1: Automatic Setup (Recommended)

Just run this one command in an empty directory:

```bash
npx create-test-project
```

Or manually:

```bash
# 1. Clone the repository
git clone https://github.com/srqai/test-sdk.git
cd test-sdk

# 2. Build the SDK
npm install
npm run build

# 3. Create a test project in a new directory
cd ..
mkdir my-test-project
cd my-test-project

# 4. Create package.json
cat > package.json << 'EOF'
{
  "name": "my-test-project",
  "version": "1.0.0",
  "type": "module",
  "scripts": {
    "test": "ts-node test.ts"
  },
  "devDependencies": {
    "@types/node": "^24.10.1",
    "ts-node": "^10.9.2",
    "typescript": "^5.9.3"
  },
  "dependencies": {
    "@authlete/typescript-sdk": "file:../test-sdk"
  }
}
EOF

# 5. Install dependencies
npm install

# 6. Create test file
cat > test.ts << 'EOF'
import { Authlete } from "@authlete/typescript-sdk";

const authlete = new Authlete({
    serverURL: "https://api.authlete.com",
    serviceId: "YOUR_SERVICE_ID",
});

console.log("âœ… SDK initialized with serviceId:", authlete.getServiceId());
EOF

# 7. Run test
npm run test
```

### Option 2: Use the Setup Script

We've included a setup script that does this automatically.

## What Gets Created

- `package.json` - Defines your project and dependencies
- `package-lock.json` - Auto-generated by npm (don't edit manually)
- `node_modules/` - Auto-generated by npm (don't edit manually)

## Notes

- `package.json` - **You create this** (or use our script)
- `package-lock.json` - **Auto-generated** by `npm install` (don't create manually)
- `node_modules/` - **Auto-generated** by `npm install` (don't create manually)
