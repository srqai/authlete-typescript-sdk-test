# ClientType

The client type, either `CONFIDENTIAL` or `PUBLIC`. See [RFC 6749, 2.1. Client Types](https://datatracker.ietf.org/doc/html/rfc6749#section-2.1)
for details.


## Example Usage

```typescript
import { ClientType } from "authlete/models";

let value: ClientType = "PUBLIC";
```

## Values

```typescript
"PUBLIC" | "CONFIDENTIAL"
```
